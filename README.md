# GeekCode Cloud IDE

## 0. 什么是Cloud IDE
Cloud IDE是基于云的集成开发环境。IDE主要工作是编写、运行、调试和打包代码。传统的方式是开发者在本地的电脑上设置IDE，但Cloud IDE 允许开发者使用浏览器设置云开发环境，在浏览器中完成开发程序。

Cloud IDEs are integrated development envronments based in the cloud. IDEs let you write run and debug your code. Typicaly developers set up IDEs on their local machine, but Cloud-based IDEs allow you to develop software with just a browser.
